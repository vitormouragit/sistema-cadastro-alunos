## 📚 Sistema de Cadastro de Alunos em C

Projeto desenvolvido em **Linguagem C** com o objetivo de realizar o cadastro e gerenciamento de alunos de forma simples e eficiente, utilizando manipulação de arquivos para persistência de dados.

---

## 🚀 Funcionalidades

* 🔐 Sistema de **login com usuário e senha**
* 👤 **Cadastro de novos usuários**
* 🎓 **Cadastro de alunos**
* 📋 **Listagem de alunos**:

  * Todos
  * Ativos
  * Inativos
* 🔎 **Ordenação de alunos**:

  * Por matrícula
  * Por nome
* ✏️ **Atualização de dados cadastrais**
* ❌ **Remoção de alunos**
* 💾 **Armazenamento em arquivos `.txt`**

---

## 🛠️ Tecnologias e Conceitos Utilizados

* Linguagem **C**
* Manipulação de arquivos (`fopen`, `fprintf`, `fscanf`, etc.)
* Uso de **structs**
* Modularização com **funções**
* Algoritmo de ordenação **Bubble Sort**
